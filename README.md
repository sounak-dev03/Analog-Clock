# Analog Clock

A simple Analog Clock built using HTML, CSS, and JavaScript.

## Features
- Real-time clock movement
- Responsive design
- Smooth hand rotation

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure

```bash
Analog-Clock/
├── Analog_Clock.html
├── Analog_Clock.css
├── Analog_Clock.js
└── Clock.jpg
```

## How to Run

1. Clone the repository 

```bash
git clone https://github.com/sounak-dev03/Analog-Clock.git
```

2. Open `Analog_Clock.html` in your browser
